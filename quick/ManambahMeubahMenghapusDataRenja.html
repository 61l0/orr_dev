Renja merupakan Data yang berisikan Rencana Kerja Setiap Direktorat / Bagian , proses pengisian data ini dilakukan guna melihat kebutuhan        setiap direktorat / Bagian .



 1. Menambah Data RENJA




   1.        Masuk Ke Menu "RENCANA KERJA".

   2.        Klik Menu "Data Renja".

   3.        Klik Tombol "Tambah Data" di Pojok Kanan


Gambar Tambah Data Renja




  4.        Isi Kolom Kolom yang Disediakan.


Gambar Form Tambah Data Renja




   5.        Klik Tombol "Simpan"  Bila Pengisian Kolom Sudah Selesai.


Gambar Form Update Data Renja




   6.        Klik Tombol "Ya" Ketika ada Notifikasi.


       Notifikasi Penyimpanan




  2. Meubah Data Renja

 


     1.        Klilk Tombol Edit Pada "Data Renja"

       

Gamabar Menu Ubah



  2        Isi / Ubah Kolom Yang Ingin Diubah.

  3.        Klik Tombol "Ya".


Gambar Kolom Form Ubah Indikator




  4.        Klik Tombol Ya Pada Notifikasi
