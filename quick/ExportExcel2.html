       Proses Export Excel merupakan hasil akhir , ayang akan digunakan sebagai laporan untuk mengetahui seluruh data Renja yang tersedia , tahapan melakukan proses export ini dapat dilihat melalui tahapan di bawah ini ....


       1.        Masuk Ke Menu Rencana Kerja

2.        Kemudian Klik Menu Data Renja.

       3.        Klik Menu Export to Excel.


       

Gambar Menu Data Renja


       4.        Klik Tombol Export To Excel

               

               



       5.        File Excel akan terdownload ke Lokal Komputer Anda.
